## 了解JavaScript
- JavaScript在1995年由Brendan Eich发明-并于1997年成为了ECMA标准
- 函数优先的轻量级、解释型(即时编译)的编程语言
--- 
## 主要功能
1. 嵌入动态文本于HTML页面
2. 对浏览器时间做出响应
3. 读写DOM元素
4. 在数据被提交到服务器前验证数据
5. 检测用户的浏览器信息
6. 基于Node.js技术进行服务器端的编程
--- 
## JavaScript语言组成
1. ECMAScript：描述该语言的语法以及基本对象 
2. DOM：（document对象模型）描述处理网页内容的方法和接口
3. BOM：（浏览器对象模型）描述浏览器进行交互的方法和接口
--- 
## JavaScript特点
- 脚本语言
- 基于对象
- 动态性
- 跨平台性
